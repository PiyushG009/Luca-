# 🎧 Luca - Music Player

Luca is a modern and responsive web-based music player inspired by Spotify.
It delivers a clean UI, smooth interactions, and an enjoyable listening experience.

---

## 🚀 Features

* 🎵 Play / Pause Music
* ⏭ Next / Previous Track
* 🎚 Progress Bar Control
* 🔊 Volume Control *(if added)*
* 📱 Responsive Design (Mobile + Desktop)
* 🎨 Clean & Modern UI

 Tech Stack

* HTML5
* CSS3
* JavaScript
